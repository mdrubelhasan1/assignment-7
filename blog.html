Answer: If we want some data on the browser, we often utilize the local storage object. session storage is used if we want to delete the data if the user closes that particular tab or the season. Cookies are used if we want them on the server.

Question No-3: what is event loop and what is the purpose of event loop?

Answer: An event loop is a programming construct that waits for and dispatches events or messages in a program.

(a) Wait for Events: The event loop waits for events to occur.
(b) Dispatch Events: When an event occurs, the event loop dispatches it to the appropriate handler or callback function.
(c) Execute Callbacks: The handler or callback function associated with the event is executed. This is where the specific logic related to the event is implemented.
(d) Repeat: The event loop continues to wait for and dispatch events, repeating the process.
